role="main">/n			<a id="top"></a>/n/n							<div id="siteNotice" class="mw-body-content"><!-- CentralNotice --></div>/n						<div class="mw-indicators mw-body-content">/n</div>/n			<h1 id="firstHeading" class="firstHeading" lang="en">Measuring programming language popularity</h1>/n									<div id="bodyContent" class="mw-body-content">/n									<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>/n								<div id="contentSub"></div>/n												<div id="jump-to-nav" class="mw-jump">/n					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>/n				</div>/n				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p>It is difficult to determine which <a href="/wiki/Programming_language" title="Programming language">programming languages</a> are most widely used,<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (October 2016)">citation needed</span></a></i>]</sup> and what usage means varies by context. One language may occupy the greater number of programmer hours, a different one have more lines of code, a third may utilize the most CPU time, and so on. Some languages are very popular for particular kinds of applications. For example, <a href="/wiki/COBOL" title="COBOL">COBOL</a> is still strong in the corporate data center, often on large <a href="/wiki/Mainframe_computer" title="Mainframe computer">mainframes</a>; <a href="/wiki/Fortran" title="Fortran">Fortran</a> in engineering applications; <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a> in embedded applications and operating systems; and other languages are regularly used to write many different kinds of applications.</p>/n<h2><span class="mw-headline" id="Methods">Methods</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Measuring_programming_language_popularity&amp;action=edit&amp;section=1" title="Edit section: Methods">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>Various methods of measuring language popularity, each subject to a different bias over what is measured, have been proposed:</p>/n<ul>/n<li>counting the number of times the language name is mentioned in web searches, such as is done by <a href="/wiki/Google_Trends" title="Google Trends">Google Trends</a></li>/n<li>counting the number of job advertisements that mention the language<sup id="cite_ref-computerweekly_1-0" class="reference"><a href="#cite_note-computerweekly-1">[1]</a></sup><sup id="cite_ref-JobsTractor_2-0" class="reference"><a href="#cite_note-JobsTractor-2">[2]</a></sup></li>/n<li>the number of books sold that teach or describe the language<sup id="cite_ref-O.27Reilly_3-0" class="reference"><a href="#cite_note-O.27Reilly-3">[3]</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup></li>/n<li>estimates of the number of existing <a href="/wiki/Lines_of_code" class="mw-redirect" title="Lines of code">lines of code</a> written in the language&#160;–  which may underestimate languages not often found in public searches<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></li>/n<li>counts of language references (i.e., to the name of the language) found using a <a href="/wiki/Web_search_engine" title="Web search engine">web search engine</a><sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup></li>/n<li>counting the number of projects in that language on <a href="/wiki/SourceForge" title="SourceForge">SourceForge</a>,<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup><sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup> and <a href="/wiki/GitHub" title="GitHub">GitHub</a><sup id="cite_ref-9" class="reference"><a href="#cite_note-9">[9]</a></sup></li>/n<li>counting the number of postings in <a href="/wiki/Usenet_newsgroup" title="Usenet newsgroup">Usenet newsgroups</a> about the language<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup></li>/n<li>comparing the number of <a href="/wiki/Commit_(version_control)" title="Commit (version control)">commits</a> or changed <a href="/wiki/Source_lines_of_code" title="Source lines of code">source lines</a> for <a href="/wiki/Open-source_model" title="Open-source model">open source</a> projects on <a href="/wiki/Open_Hub" title="Open Hub">Open Hub</a><sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup></li>/n</ul>/n<h2><span class="mw-headline" id="Indices">Indices</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Measuring_programming_language_popularity&amp;action=edit&amp;section=2" title="Edit section: Indices">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>Several indices have been published:</p>/n<ul>/n<li>The monthly <i><a href="/wiki/Tiobe_index" class="mw-redirect" title="Tiobe index">TIOBE Programming Community Index</a></i> has been published since 2001, and shows the top 10 languages' popularity graphically, the top 20 languages with a rating and delta, and the top 50 languages' ratings.<sup id="cite_ref-tiobe_12-0" class="reference"><a href="#cite_note-tiobe-12">[12]</a></sup> The numbers are based on searching the Web with certain phrases that include language names and counting the numbers of hits returned.</li>/n<li>The <i>PYPL PopularitY of Programming Language</i><sup id="cite_ref-pypl_13-0" class="reference"><a href="#cite_note-pypl-13">[13]</a></sup> is an indicator based on <a href="/wiki/Google_Trends" title="Google Trends">Google Trends</a>, reflecting the developers' searches for "&lt;programming language&gt; tutorial", instead of what pages are available.<sup id="cite_ref-pypl_13-1" class="reference"><a href="#cite_note-pypl-13">[13]</a></sup> It shows the popularity trends since 2004, worldwide or separated for 5 countries.</li>/n<li>The <i>RedMonk Programming Language Rankings</i><sup id="cite_ref-14" class="reference"><a href="#cite_note-14">[14]</a></sup> are derived from a correlation of programming traction on <a href="/wiki/GitHub" title="GitHub">GitHub</a> (usage) and <a href="/wiki/Stack_Overflow" title="Stack Overflow">Stack Overflow</a> (discussion).</li>/n<li>The <i>Trendy Skills</i><sup id="cite_ref-15" class="reference"><a href="#cite_note-15">[15]</a></sup> searches and extracts from popular advertising websites the skills and technologies that employers are looking and classifies skills sought in categories, one of which is the Programming Languages category. It allows the user to see the trends for one or more skills or categories at specified time ranges. Data is also accessible via a public <a href="/wiki/API" class="mw-redirect" title="API">API</a>, so anyone can generate their own statistics.</li>/n<li><i><a href="/wiki/Indeed" class="mw-redirect" title="Indeed">Indeed</a> 2016 survey</i>. Results show that among job advertisements <a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a> is more popular than other languages combined.<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">[16]</a></sup></li>/n<li><a href="/wiki/Stack_Overflow" title="Stack Overflow">Stack Overflow</a>'s 2016 <i>Developer Survey Results</i>. According to poll <a href="/wiki/JavaScript" title="JavaScript">JavaScript</a> is used by 55% of developers.<sup id="cite_ref-17" class="reference"><a href="#cite_note-17">[17]</a></sup></li>/n<li><i>Krihelinator.xyz</i><sup id="cite_ref-18" class="reference"><a href="#cite_note-18">[18]</a></sup> ranks programming languages based on their github contribution rate according to this <i>formula</i>.<sup id="cite_ref-19" class="reference"><a href="#cite_note-19">[19]</a></sup></li>/n<li><a href="/wiki/IEEE_Spectrum" title="IEEE Spectrum">IEEE Spectrum</a>'s 2016 ranking of top programming languages<sup id="cite_ref-ieeerank_20-0" class="reference"><a href="#cite_note-ieeerank-20">[20]</a></sup> "synthesises 12 metrics from 10 sources to arrive at an overall ranking of language popularity".<sup id="cite_ref-ieeemethod_21-0" class="reference"><a href="#cite_note-ieeemethod-21">[21]</a></sup> The various metrics were collected from <a href="/wiki/GitHub" title="GitHub">GitHub</a>, <a href="/wiki/Google_Search" title="Google Search">Google Search</a> and <a href="/wiki/Google_Trends" title="Google Trends">Trends</a>, <a href="/wiki/Twitter" title="Twitter">Twitter</a>, <a href="/wiki/Stack_Overflow" title="Stack Overflow">Stack Overflow</a>, <a href="/wiki/Reddit" title="Reddit">Reddit</a>, <a href="/wiki/Hacker_News" title="Hacker News">Hacker News</a>, <a href="/wiki/Career_Builder" class="mw-redirect" title="Career Builder">Career Builder</a>, <a href="/wiki/Dice.com" title="Dice.com">Dice.com</a>, and <a href="/wiki/IEEE_Xplore" title="IEEE Xplore">IEEE Xplore Digital Library</a>. The interactive ranking app<sup id="cite_ref-ieeeapp_22-0" class="reference"><a href="#cite_note-ieeeapp-22">[22]</a></sup> allows adjustment of each metric's weight, and also filtering languages by "type" (Web, Mobile, Enterprise, Embedded).</li>/n</ul>/n<h2><span class="mw-headline" id="References">References</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Measuring_programming_language_popularity&amp;action=edit&amp;section=3" title="Edit section: References">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div class="reflist columns references-column-width" style="-moz-column-width: 30em; -webkit-column-width: 30em; column-width: 30em; list-style-type: decimal;">/n<ol class="references">/n<li id="cite_note-computerweekly-1"><span class="mw-cite-backlink"><b><a href="#cite_ref-computerweekly_1-0">^</a></b></span> <span class="reference-text"><cite class="citation web"><a rel="nofollow" class="external text" href="http://www.computerweekly.com/Articles/2007/09/11/226631/sslcomputer-weekly-it-salary-survey-finance-boom-drives-it-job.htm">"SSL/Computer Weekly IT salary survey: finance boom drives IT job growth"</a>.
