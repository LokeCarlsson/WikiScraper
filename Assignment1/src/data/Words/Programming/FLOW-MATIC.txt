class="mw-parser-output"><table class="infobox vevent" style="width:22em">/n<caption class="summary">FLOW-MATIC</caption>/n<tr>/n<th scope="row"><a href="/wiki/Programming_paradigm" title="Programming href="/wiki/Computing_platform" title="Computing platform">Platform</a></th>/n<td><a href="/wiki/UNIVAC_I" title="UNIVAC I">UNIVAC I</a></td>/n</tr>/n<tr>/n<th colspan="2" style="text-align:center;background-color: #eee;">Influenced by</th>/n</tr>/n<tr>/n<td colspan="2" style="text-align:center"><a href="/wiki/ARITH-MATIC" title="ARITH-MATIC">ARITH-MATIC</a></td>/n</tr>/n<tr>/n<th colspan="2" style="text-align:center;background-color: #eee;">Influenced</th>/n</tr>/n<tr>/n<td colspan="2" style="text-align:center"><a href="/wiki/MATH-MATIC" title="MATH-MATIC">MATH-MATIC</a>, <a href="/wiki/AIMACO" title="AIMACO">AIMACO</a>, <a href="/wiki/COBOL" title="COBOL">COBOL</a></td>/n</tr>/n</table>/n<p><b>FLOW-MATIC</b>, originally known as <b>B-0</b> (<b>Business Language version 0</b>), was the first English-like <a href="/wiki/Computer_data_processing" class="mw-redirect" title="Computer data processing">data processing</a> language. It was developed for the <a href="/wiki/UNIVAC_I" title="UNIVAC I">UNIVAC I</a> at <a href="/wiki/Remington_Rand" title="Remington Rand">Remington Rand</a> under <a href="/wiki/Grace_Hopper" title="Grace Hopper">Grace Hopper</a> during the period from 1955 until 1959. It had a strong influence on the development of <a href="/wiki/COBOL" title="COBOL">COBOL</a>.</p>/n<p></p>/n<div id="toc" class="toc">/n<div class="toctitle">/n<h2>Contents</h2>/n</div>/n<ul>/n<li class="toclevel-1 tocsection-1"><a href="#Development"><span class="tocnumber">1</span> <span class="toctext">Development</span></a></li>/n<li class="toclevel-1 tocsection-2"><a href="#Innovations_and_influence"><span class="tocnumber">2</span> <span class="toctext">Innovations and influence</span></a></li>/n<li class="toclevel-1 tocsection-3"><a href="#Sample_program"><span class="tocnumber">3</span> <span class="toctext">Sample program</span></a></li>/n<li class="toclevel-1 tocsection-4"><a href="#Notes"><span class="tocnumber">4</span> <span class="toctext">Notes</span></a></li>/n<li class="toclevel-1 tocsection-5"><a href="#References"><span class="tocnumber">5</span> <span class="toctext">References</span></a></li>/n</ul>/n</div>/n<p></p>/n<h2><span class="mw-headline" id="Development">Development</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FLOW-MATIC&amp;action=edit&amp;section=1" title="Edit section: Development">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>Hopper had found that business data processing customers were uncomfortable with mathematical notation.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> In late 1953 she proposed that data processing problems should be expressed using English keywords, but Rand management considered the idea unfeasible. In early 1955, she and her team wrote a specification for such a programming language and implemented a prototype.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup> The FLOW-MATIC compiler became publicly available in early 1958 and was substantially complete in 1959.<sup id="cite_ref-sammet316_4-0" class="reference"><a href="#cite_note-sammet316-4">[4]</a></sup></p>/n<h2><span class="mw-headline" id="Innovations_and_influence">Innovations and influence</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FLOW-MATIC&amp;action=edit&amp;section=2" title="Edit section: Innovations and influence">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>First, FLOW-MATIC was the first programming language to <a href="/wiki/English-like_programming_language" class="mw-redirect" title="English-like programming language">express operations using English-like statements</a>.<sup id="cite_ref-sammet316_4-1" class="reference"><a href="#cite_note-sammet316-4">[4]</a></sup></p>/n<p>Second, FLOW-MATIC was the first system to distinctly separate the description of data from the operations on it. Its <a href="/wiki/Data_definition_language" title="Data definition language">data definition language</a>, unlike its executable statements, was not English-like; rather, data structures were defined by filling in pre-printed forms.<sup id="cite_ref-sammet316_4-2" class="reference"><a href="#cite_note-sammet316-4">[4]</a></sup></p>/n<p>Flow-Matic was a major influence in the design of <a href="/wiki/COBOL" title="COBOL">COBOL</a>, since only it and its direct descendent <a href="/wiki/AIMACO" title="AIMACO">AIMACO</a> were in actual use at the time.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup> Several elements of Flow-Matic were incorporated into COBOL:</p>/n<ul>/n<li>Defining files in advance, and separating into <code class="mw-highlight" id="" style="" dir="ltr">INPUT</code> and <code class="mw-highlight" id="" style="" dir="ltr">OUTPUT</code> files.</li>/n<li>Qualification of data-names (<code class="mw-highlight" id="" style="" dir="ltr">IN</code> or <code class="mw-highlight" id="" style="" dir="ltr">OF</code> clause).</li>/n<li><code class="mw-highlight" id="" style="" dir="ltr">IF END OF DATA (AT END)</code> clause on file <code class="mw-highlight" id="" style="" dir="ltr">READ</code> operations.</li>/n<li>Figurative constant <code class="mw-highlight" id="" style="" dir="ltr">ZERO</code> (originally <code class="mw-highlight" id="" style="" dir="ltr">ZZZ...ZZZ</code>, where number of <code class="mw-highlight" id="" style="" dir="ltr">Z</code>'s indicated precision).</li>/n<li>Dividing the program into sections, separating different parts of the program. Flow-Matic sections included <b><code class="mw-highlight" id="" style="" dir="ltr">Computer</code></b> (Environment Division), <b><code class="mw-highlight" id="" style="" dir="ltr">Directory</code></b> (Data Division), and <b><code class="mw-highlight" id="" style="" dir="ltr">Compiler</code></b> (Procedure Division).</li>/n</ul>/n<h2><span class="mw-headline" id="Sample_program">Sample program</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FLOW-MATIC&amp;action=edit&amp;section=3" title="Edit section: Sample program">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>A sample FLOW-MATIC program:<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup><sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup></p>/n<pre>/n (0)  INPUT INVENTORY FILE-A PRICE FILE-B ; OUTPUT PRICED-INV FILE-C UNPRICED-INV/n     FILE-D ; HSP D ./n (1)  COMPARE PRODUCT-NO (A) WITH PRODUCT-NO (B) ; IF GREATER GO TO OPERATION 10 ;/n     IF EQUAL GO TO OPERATION 5 ; OTHERWISE GO TO OPERATION 2 ./n (2)  TRANSFER A TO D ./n (3)  WRITE-ITEM D ./n (4)  JUMP TO OPERATION 8 ./n (5)  TRANSFER A TO C ./n (6)  MOVE UNIT-PRICE (B) TO UNIT-PRICE (C) ./n (7)  WRITE-ITEM C ./n (8)  READ-ITEM A ; IF END OF DATA GO TO OPERATION 14 ./n (9)  JUMP TO OPERATION 1 ./n(10)  READ-ITEM B ; IF END OF DATA GO TO OPERATION 12 ./n(11)  JUMP TO OPERATION 1 ./n(12)  SET OPERATION 9 TO GO TO OPERATION 2 ./n(13)  JUMP TO OPERATION 2 ./n(14)  TEST PRODUCT-NO (B) AGAINST ; IF EQUAL GO TO OPERATION 16 ;/n     OTHERWISE GO TO OPERATION 15 ./n(15)  REWIND B ./n(16)  CLOSE-OUT FILES C ; D ./n(17)  STOP . (END)/n</pre>/n<p>Note that this sample includes only the executable statements of the program, the <code class="mw-highlight" id="" style="" dir="ltr">COMPILER</code> section. The record fields <code class="mw-highlight" id="" style="" dir="ltr">PRODUCT-NO</code> and <code class="mw-highlight" id="" style="" dir="ltr">UNIT-PRICE</code> would have been defined in the <code class="mw-highlight" id="" style="" dir="ltr">DIRECTORY</code> section, which did not use English-like syntax.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup></p>/n<h2><span class="mw-headline" id="Notes">Notes</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=FLOW-MATIC&amp;action=edit&amp;section=4" title="Edit section: Notes">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div class="reflist columns references-column-width" style="-moz-column-width: 30em; -webkit-column-width: 30em; column-width: 30em; list-style-type: decimal;">/n<ol class="references">/n<li id="cite_note-1"><span class="mw-cite-backlink"><b><a href="#cite_ref-1">^</a></b></span> <span class="reference-text">Hopper (1959) p.&#160;198.</span></li>/n<li id="cite_note-2"><span class="mw-cite-backlink"><b><a href="#cite_ref-2">^</a></b></span> <span class="reference-text">“I used to be a mathematics professor.
