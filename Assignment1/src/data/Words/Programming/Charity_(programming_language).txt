/>.html</a></span></td>/n</tr>/n</table>/n<p><b>Charity</b> is an experimental <a href="/wiki/Purely_functional_programming" title="Purely functional programming">purely functional</a> <a href="/wiki/Programming_language" title="Programming language">programming language</a>, developed at the <a href="/wiki/University_of_Calgary" title="University of Calgary">University of Calgary</a> under the supervision of Robin Cockett. Based on ideas by <a rel="nofollow" class="external text" href="https://web.archive.org/web/20080603005743/http://www.tom.sfc.keio.ac.jp/~hagino/index.html.en">Hagino Tatsuya</a>, it is completely grounded in <a href="/wiki/Category_theory" title="Category theory">category theory</a>.</p>/n<p>Disregarding <a href="/wiki/Input/output" title="Input/output">interactions with the outside world</a>, all Charity programs are guaranteed to terminate or stay productive.</p>/n<p>The language allows ordinary recursive data types, such as might be found in <a href="/wiki/ML_(programming_language)" title="ML (programming language)">ML</a>, which are required to be finite, and corecursive data types, which are allowed to be potentially infinite. The control structure for operating on recursive data types is primitive recursion or <a href="/wiki/Paramorphism" title="Paramorphism">paramorphism</a>, and the control structure for corecursive data types is primitive co-recursion or <a href="/wiki/Apomorphism" title="Apomorphism">apomorphism</a>. Neither control structure can operate over the other kind of data, so all paramorphisms terminate and all apomorphisms are productive.</p>/n<h2><span class="mw-headline" id="References">References</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Charity_(programming_language)&amp;action=edit&amp;section=1" title="Edit section: References">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div class="mw-references-wrap">/n<ol class="references">/n<li id="cite_note-1"><span class="mw-cite-backlink"><b><a href="#cite_ref-1">^</a></b></span> <span class="reference-text"><cite class="citation journal">Cockett, Robin; Fukushima, Tom (May 27, 1992). <a rel="nofollow" class="external text" href="ftp://ftp.cpsc.ucalgary.ca/pub/projects/charity/literature/papers_and_reports/about_charity.ps">"About Charity"</a>. <i>Yellow Series Report</i>.
