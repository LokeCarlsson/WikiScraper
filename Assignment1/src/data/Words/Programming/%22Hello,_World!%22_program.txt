Being a very simple program in most <a href="/wiki/Programming_language" title="Programming language">programming languages</a>, it is often used to illustrate the basic <a href="/wiki/Syntax" title="Syntax">syntax</a> of a programming language for a working program.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> It is often the very first program people write class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%22Hello,_World!%22_program&amp;action=edit&amp;section=1" title="Edit section: Purpose">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>A "Hello, World!" program is traditionally used to introduce novice programmers to a programming language.</p>/n<p>"Hello, world!" is also traditionally used in a <a href="/wiki/Sanity_test" class="mw-redirect" title="Sanity test">sanity test</a> to make sure that a computer language is correctly installed, and that the operator understands how to use it.</p>/n<p>"Hello, world!" is also used by computer hackers as a <a href="/wiki/Proof_of_concept" title="Proof of concept">proof of concept</a> that arbitrary code can be executed through an <a href="/wiki/Exploit_(computer_security)" title="Exploit (computer security)">exploit</a> where the system designers did not intend code to be executed.</p>/n<h2><span href="/wiki/Brian_Kernighan" title="Brian Kernighan">Brian Kernighan</a> (1978)</div>/n</div>/n</div>/n<p>While small test programs have existed since the development of programmable <a href="/wiki/Computer" title="Computer">computers</a>, the tradition of using the phrase "Hello, world!" as a test message was influenced by an example program in the seminal book <i><a href="/wiki/The_C_Programming_Language_(book)" class="mw-redirect" title="The C Programming Language (book)">The C Programming Language</a></i>.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> The example program from that book prints "<code>hello,&#160;world</code>" (without capital letters or exclamation mark), and was inherited<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (March 2015)">citation needed</span></a></i>]</sup> from a 1974 <a href="/wiki/Bell_Labs" title="Bell Labs">Bell Laboratories</a> internal memorandum by <a href="/wiki/Brian_Kernighan" title="Brian Kernighan">Brian Kernighan</a>, <i>Programming in C.A Tutorial</i>:<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup></p>/n<div class="mw-highlight mw-content-ltr" dir="ltr">/n<pre>/n<span class="cp">#include</span> <span class="cpf">&lt;stdio.h&gt;</span>/n/n<span class="n">main</span><span class="p">(</span> <span class="p">)</span>/n<span class="p">{</span>/n        <span class="n">printf</span><span class="p">(</span><span class="s">"hello, world</span><span class="se">\n</span><span class="s">"</span><span class="p">);</span>/n<span class="p">}</span>/n</pre></div>/n<p>The C version was preceded by Kernighan's own 1972 <i>A Tutorial Introduction to the Language <a href="/wiki/B_(programming_language)" title="B (programming language)">B</a></i>,<sup id="cite_ref-langb_4-0" class="reference"><a href="#cite_note-langb-4">[4]</a></sup> where the first known version of the program is found in an example used to illustrate external variables:</p>/n<div class="mw-highlight mw-content-ltr" dir="ltr">/n<pre>/nmain(){/n  extrn a,b,c;/n  putchar(a); putchar(b); putchar(c); putchar('!*n');/n  }/n/na 'hell';/nb 'o, w';/nc 'orld';/n</pre></div>/n<p>The program prints <i>hello,&#160;world!</i> on the terminal, including a <a href="/wiki/Newline" title="Newline">newline</a> character. The phrase is divided into multiple variables because in B, a character constant is limited to four <a href="/wiki/ASCII" title="ASCII">ASCII</a> characters. The previous example in the tutorial printed <i>hi!</i> on the terminal, and the phrase <i>hello,&#160;world!</i> was introduced as a slightly longer greeting that required several character constants for its expression.</p>/n<p>It is also claimed that<sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Manual_of_Style/Words_to_watch#Unsupported_attributions" title="Wikipedia:Manual of Style/Words to watch"><span title="The material near this tag may use weasel words or too-vague attribution. (March 2015)">by whom?</span></a></i>]</sup> <i>hello, world</i> originated instead with <a href="/wiki/BCPL" title="BCPL">BCPL</a> (1967).<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup><sup class="noprint Inline-Template" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Identifying_reliable_sources" title="Wikipedia:Identifying reliable sources"><span title="The material near this tag may rely on an unreliable source. (March 2015)">unreliable source?</span></a></i>]</sup>This claim is supported by the archived notes of the inventors of BCPL, Prof. Brian Kernighan at Princeton and Martin Richards at Cambridge.<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (May 2016)">citation needed</span></a></i>]</sup></p>/n<p>For modern languages, hello, world programs vary in sophistication. For example, the <a href="/wiki/Go_programming_language" class="mw-redirect" title="Go programming language">Go programming language</a> introduced a multilingual program,<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup> <a href="/wiki/Sun_Microsystems" title="Sun Microsystems">Sun</a> demonstrated a <a href="/wiki/Java_(programming_language)" title="Java (programming language)">Java</a> hello, world based on <a href="/wiki/Scalable_vector_graphics" class="mw-redirect" title="Scalable vector graphics">scalable vector graphics</a>,<sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup> and the <a href="/wiki/XL_(programming_language)" title="XL (programming language)">XL</a> programming language features a spinning Earth hello, world using 3D graphics.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup> While some languages such as <a href="/wiki/Perl_(programming_language)" class="mw-redirect" title="Perl (programming language)">Perl</a>, <a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a> or <a href="/wiki/Ruby_(programming_language)" title="Ruby (programming language)">Ruby</a> may need only a single statement to print "hello, world", a low-level <a href="/wiki/Assembly_language" title="Assembly language">assembly language</a> may require dozens of commands. Mark Guzdial and Elliot Soloway have suggested that the "hello, world" test message may be outdated now that graphics and sound can be manipulated as easily as text.<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">[9]</a></sup></p>/n<h2><span class="mw-headline" id="Variations">Variations</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%22Hello,_World!%22_program&amp;action=edit&amp;section=3" title="Edit section: Variations">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div class="thumb tleft">/n<div class="thumbinner" style="width:222px;"><a href="/wiki/File:PSP-Homebrew.jpeg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8c/PSP-Homebrew.jpeg/220px-PSP-Homebrew.jpeg" width="220" height="165" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/8c/PSP-Homebrew.jpeg/330px-PSP-Homebrew.jpeg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/8c/PSP-Homebrew.jpeg/440px-PSP-Homebrew.jpeg 2x" data-file-width="2816" data-file-height="2112" /></a>/n<div class="thumbcaption">/n<div class="magnify"><a href="/wiki/File:PSP-Homebrew.jpeg" class="internal" title="Enlarge"></a></div>/nA "Hello, world!" program running on Sony's <a href="/wiki/PlayStation_Portable_homebrew" title="PlayStation Portable homebrew">PlayStation Portable</a> as a proof of concept.</div>/n</div>/n</div>/n<p>There are many variations on the punctuation and casing of the phrase. Variations include the presence or absence of the comma and exclamation mark, and the capitalization of the 'H', both the 'H' and the 'W', or neither. Some languages are forced to implement different forms, such as "HELLO WORLD", on systems that support only capital letters, while many "hello, world" programs in <a href="/wiki/Esoteric_programming_language" title="Esoteric programming language">esoteric languages</a> print out a slightly modified string. For example, the first non-trivial <a href="/wiki/Malbolge" title="Malbolge">Malbolge</a> program printed "HEllO WORld", this having been determined to be <a href="/wiki/Principle_of_good_enough" title="Principle of good enough">good enough</a>.<sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup></p>/n<p>There are variations in spirit, as well. <a href="/wiki/Functional_programming" title="Functional programming">Functional programming</a> languages, like <a href="/wiki/Lisp_(programming_language)" title="Lisp (programming language)">Lisp</a>, <a href="/wiki/ML_(programming_language)" title="ML (programming language)">ML</a> and <a href="/wiki/Haskell_(programming_language)" title="Haskell (programming language)">Haskell</a>, tend to substitute a <a href="/wiki/Factorial" title="Factorial">factorial</a> program for Hello, World, as functional programming emphasizes recursive techniques, whereas the original examples emphasize I/O, which violates the spirit of pure functional programming by producing side effects. Languages otherwise capable of Hello, World (Assembly, C, <a href="/wiki/VHDL" title="VHDL">VHDL</a>) may also be used in <a href="/wiki/Embedded_system" title="Embedded system">embedded systems</a>, where text output is either difficult (requiring additional components or communication with another computer) or nonexistent. For devices such as <a href="/wiki/Microcontroller" title="Microcontroller">microcontrollers</a>, <a href="/wiki/Field-programmable_gate_arrays" class="mw-redirect" title="Field-programmable gate arrays">field-programmable gate arrays</a>, and <a href="/wiki/CPLD" class="mw-redirect" title="CPLD">CPLD</a>'s, "Hello, World" may thus be substituted with a blinking <a href="/wiki/Light-emitting_diode" title="Light-emitting diode">LED</a>, which demonstrates timing and interaction between components.<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup><sup id="cite_ref-12" class="reference"><a href="#cite_note-12">[12]</a></sup><sup id="cite_ref-13" class="reference"><a href="#cite_note-13">[13]</a></sup><sup id="cite_ref-14" class="reference"><a href="#cite_note-14">[14]</a></sup><sup id="cite_ref-15" class="reference"><a href="#cite_note-15">[15]</a></sup></p>/n<p>The <a href="/wiki/Debian" title="Debian">Debian</a> and <a href="/wiki/Ubuntu_(operating_system)" title="Ubuntu (operating system)">Ubuntu</a> <a href="/wiki/Linux" title="Linux">Linux</a> distributions provide the "hello, world" program through the <a href="/wiki/Advanced_Packaging_Tool" class="mw-redirect" title="Advanced Packaging Tool">apt</a> packaging system; this allows users to simply type "apt-get install hello" for the program to be installed, along with any <a href="/wiki/Dependency_(software)" class="mw-redirect" title="Dependency (software)">software dependencies</a>. While of itself useless, it serves as a <a href="/wiki/Sanity_check" title="Sanity check">sanity check</a> and a simple example to newcomers of how to install a package.
