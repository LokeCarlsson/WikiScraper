class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Non-English-based_programming_languages&amp;action=edit&amp;section=1" title="Edit section: Prevalence of English-based programming languages">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div role="note" class="hatnote navigation-not-searchable">Further information: <a href="/wiki/English_in_computing" title="English in computing">English in computing</a></div>/n<p>There has been an overwhelming trend in programming languages to use the English language to inspire the choice of <a href="/wiki/Keyword_(computer_programming)" class="mw-redirect" title="Keyword (computer programming)">keywords</a> and code libraries. According to the HOPL online database of languages,<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> out of the 8,500+ programming languages recorded, roughly 2,400 of them were developed in the <a href="/wiki/United_States" title="United States">United States</a>, 600 in the <a href="/wiki/United_Kingdom" title="United Kingdom">United Kingdom</a>, 160 in Canada, and 75 in Australia.</p>/n<p>In other words, over a third of all programming languages were developed in a country with <a href="/wiki/English_language" title="English language">English</a> as the primary language. This does not take into account the usage share of each language, situations where a language was developed in a non-English-speaking country but used English to appeal to an international audience (see the case of <a href="/wiki/Python_(programming_language)" title="Python (programming language)">Python</a> from the <a href="/wiki/Netherlands" title="Netherlands">Netherlands</a>, <a href="/wiki/Ruby_(programming_language)" title="Ruby (programming language)">Ruby</a> from <a href="/wiki/Japan" title="Japan">Japan</a>, and <a href="/wiki/Lua_(programming_language)" title="Lua (programming language)">Lua</a> from <a href="/wiki/Brazil" title="Brazil">Brazil</a>), and situations where it was based on another language which used English (see the case of <a href="/wiki/Caml" title="Caml">Caml</a>, developed in France but using English keywords).</p>/n<h2><span class="mw-headline" id="International_programming_languages">International programming languages</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Non-English-based_programming_languages&amp;action=edit&amp;section=2" title="Edit section: International programming languages">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p><a href="/wiki/ALGOL_68" title="ALGOL 68">ALGOL 68</a>'s standard document was published in numerous <a href="/wiki/Natural_languages" class="mw-redirect" title="Natural languages">natural languages</a>, and the standard allowed the internationalisation of the programming language itself.</p>/n<p>On December 20, 1968, the "Final Report" (MR 101) was adopted by the Working Group, then subsequently approved by the General Assembly of <a href="/wiki/UNESCO" title="UNESCO">UNESCO</a>'s <a href="/wiki/IFIP" class="mw-redirect" title="IFIP">IFIP</a> for publication. Translations of the standard were made for <a href="/wiki/Russian_language" title="Russian language">Russian</a>, <a href="/wiki/German_language" title="German language">German</a>, <a href="/wiki/French_language" title="French language">French</a>, <a href="/wiki/Bulgarian_language" title="Bulgarian language">Bulgarian</a>, and then later <a href="/wiki/Japanese_language" title="Japanese language">Japanese</a>. The standard was made available also in <a href="/wiki/Braille" title="Braille">Braille</a>. ALGOL 68 went on to become the <a href="/wiki/GOST" title="GOST">GOST/ГОСТ</a>-27974-88 standard in the <a href="/wiki/Soviet_Union" title="Soviet Union">Soviet Union</a>.</p>/n<ul>/n<li>GOST 27974-88 Programming language ALGOL 68 - Язык программирования АЛГОЛ 68<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup></li>/n<li>GOST 27975-88 Programming language ALGOL 68 extended - Язык программирования АЛГОЛ 68 расширенный<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup></li>/n</ul>/n<p>In English, Algol68's <i>reverent</i> case statement reads <b>case</b> ~ <b>in</b> ~ <b>out</b> ~ <b>esac</b>. In <a href="/wiki/Cyrillic" class="mw-redirect" title="Cyrillic">Cyrillic</a>, this reads <b>выб</b> ~ <b>в</b> ~ <b>либо</b> ~ <b>быв</b>.</p>/n<h2><span class="mw-headline" id="Based_on_non-English_languages">Based on non-English languages</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Non-English-based_programming_languages&amp;action=edit&amp;section=3" title="Edit section: Based on non-English languages">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div role="note" class="hatnote navigation-not-searchable">See also: <a href="/wiki/List_of_Chinese_programming_languages" title="List of Chinese programming languages">List of Chinese programming languages</a></div>/n<ul>/n<li>Aheui – An <a href="/wiki/Esoteric_programming_language" title="Esoteric programming language">esoteric programming language</a> similar to <a href="/wiki/Befunge" title="Befunge">Befunge</a> but using <a href="/wiki/Hangul" title="Hangul">Hangul</a> (<a href="/wiki/Korean_language" title="Korean language">Korean</a>)<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup></li>/n<li>AMMORIA – Open source object oriented Arabic programming language, designed especially for Arabs.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></li>/n<li><a href="/w/index.php?title=Analitik&amp;action=edit&amp;redlink=1" class="new" title="Analitik (page does not exist)">Analitik</a><span class="noprint" style="font-size:85%;
