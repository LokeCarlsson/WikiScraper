href="/wiki/Page_description_language" title="Page description language">page description language</a> contains constructs intended to make it easier to write programs that control the layout of text and graphics on a page).</p>/n<p>Conversely, a <a href="/wiki/Domain-specific_programming_language" class="mw-redirect" title="Domain-specific programming language">domain-specific programming
