id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p>In <a href="/wiki/Computer_programming" title="Computer programming">computer programming</a>, a <b>scientific language</b> is a <a href="/wiki/Programming_language" title="Programming language">programming language</a> optimized for the use of mathematical <a href="/wiki/Formula" title="Formula">formulas</a> and <a href="/wiki/Matrix_(mathematics)" title="Matrix (mathematics)">matrices</a>. Although these functions can be performed using any language, they are more easily expressed in scientific languages.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup> Scientific languages include <a href="/wiki/MATLAB" title="MATLAB">MATLAB</a>, <a href="/wiki/FORTRAN" class="mw-redirect" title="FORTRAN">FORTRAN</a>, <a href="/wiki/ALGOL" title="ALGOL">ALGOL</a>, <a href="/wiki/APL_(programming_language)" title="APL (programming language)">APL</a>,<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> <a href="/wiki/J_(programming_language)" title="J (programming language)">J</a>, and <a href="/wiki/Julia_(programming_language)" title="Julia (programming language)">Julia</a>.</p>/n<p>In other fields, scientific language is loosely defined as being grammatically correct, and giving concise and correct information.</p>/n<h2><span class="mw-headline" id="See_also">See also</span><span
