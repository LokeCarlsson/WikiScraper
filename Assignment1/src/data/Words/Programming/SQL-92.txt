class="mw-body-content">/n									<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>/n								<div id="contentSub"></div>/n												<div id="jump-to-nav" class="mw-jump">/n					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>/n				</div>/n				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p><b>SQL-92</b> was the third revision of the <a href="/wiki/SQL" title="SQL">SQL</a> <a href="/wiki/Database" title="Database">database</a> <a href="/wiki/Query_language" title="Query language">query language</a>. Unlike SQL-89, it was a major revision of the standard. Aside from a few minor incompatibilities, the SQL-89 standard is forward-compatible with SQL-92.</p>/n<p>The standard specification itself grew about five times compared to SQL-89. Much of it was due to more precise specifications of existing features; the increase due to new features was only by a factor of 1.5–2. Many of the new features had already been implemented by vendors before the new standard was adopted.<sup id="cite_ref-MeltonSimon1993_1-0" class="reference"><a href="#cite_note-MeltonSimon1993-1">[1]</a></sup> However, most of the new features were added to the "intermediate" and "full" tiers of the specification, meaning that conformance with SQL-92 entry level was scarcely any more demanding than conformance with <a href="/w/index.php?title=SQL-89&amp;action=edit&amp;redlink=1" class="new" title="SQL-89 (page does not exist)">SQL-89</a>.</p>/n<p>Later revisions of the standard include <a href="/wiki/SQL:1999" title="SQL:1999">SQL:1999</a> (SQL3), <a href="/wiki/SQL:2003" title="SQL:2003">SQL:2003</a>, <a href="/wiki/SQL:2008" title="SQL:2008">SQL:2008</a>, <a href="/wiki/SQL:2011" title="SQL:2011">SQL:2011</a> and <a href="/wiki/SQL:2016" title="SQL:2016">SQL:2016</a>.</p>/n<p></p>/n<div id="toc" class="toc">/n<div class="toctitle">/n<h2>Contents</h2>/n</div>/n<ul>/n<li class="toclevel-1 tocsection-1"><a href="#New_features"><span class="tocnumber">1</span> <span class="toctext">New features</span></a></li>/n<li class="toclevel-1 tocsection-2"><a href="#Extensions"><span class="tocnumber">2</span> <span class="toctext">Extensions</span></a></li>/n<li class="toclevel-1 tocsection-3"><a href="#References"><span class="toctext">External links</span></a></li>/n</ul>/n</div>/n<p></p>/n<h2><span class="mw-headline" id="New_features">New features</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SQL-92&amp;action=edit&amp;section=1" title="Edit section: New features">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>Significant new features include:<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup></p>/n<ul>/n<li>New data types defined: <code>DATE</code>, <code>TIME</code>, <code>TIMESTAMP</code>, <code>INTERVAL</code>, <code>BIT</code> string, <code>VARCHAR</code> strings, and <code>NATIONAL CHARACTER</code> strings.</li>/n<li>Support for additional <a href="/wiki/Character_set" class="mw-redirect" title="Character set">character sets</a> beyond the base requirement for representing SQL statements.</li>/n<li>New scalar operations such as string concatenation and <a href="/wiki/Substring" title="Substring">substring</a> extraction, date and time mathematics, and conditional statements.</li>/n<li>New set operations such as <code>UNION JOIN</code>, <code>NATURAL JOIN</code>, set differences, and set intersections.</li>/n<li>Conditional expressions with <code>CASE</code>. For an example, see <a href="/wiki/Case_(SQL)" class="mw-redirect" title="Case (SQL)">Case (SQL)</a>.</li>/n<li>Support for alterations of <a href="/wiki/Data_Definition_Language" class="mw-redirect" title="Data Definition Language">schema definitions</a> via <code>ALTER</code> and <code>DROP</code>.</li>/n<li>Bindings for <a href="/wiki/C_(programming_language)" title="C (programming language)">C</a>, <a href="/wiki/Ada_(programming_language)" title="Ada (programming language)">Ada</a>, and <a href="/wiki/MUMPS" title="MUMPS">MUMPS</a>.</li>/n<li>New features for user privileges.</li>/n<li>New integrity-checking functionality such as within a <a href="/wiki/Check_Constraint" class="mw-redirect" title="Check Constraint"><code>CHECK</code> constraint</a>.</li>/n<li>A new <i>information schema</i>—read-only views about database metadata like what tables it contains, etc. For example, <code>SELECT * FROM INFORMATION_SCHEMA.TABLES;</code>.</li>/n<li>Dynamic execution of queries (as opposed to prepared).</li>/n<li>Better support for remote database access.</li>/n<li>Temporary tables; <code>CREATE TEMP TABLE</code> etc.</li>/n<li><a href="/wiki/Database_transaction" title="Database transaction">Transaction</a> <a href="/wiki/Isolation_(database_systems)" title="Isolation (database systems)">isolation levels</a>.</li>/n<li>New operations for changing data types on the fly via <code>CAST (expr AS type)</code>.</li>/n<li><a href="/wiki/Cursor_(databases)#Scrollable_cursors" title="Cursor (databases)">Scrolled cursors</a>.</li>/n<li>Compatibility flagging for backwards and forwards compatibility with other SQL standards.</li>/n</ul>/n<h2><span class="mw-headline" id="Extensions">Extensions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SQL-92&amp;action=edit&amp;section=2" title="Edit section: Extensions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<p>Two significant extensions were published after standard (but before the next major iteration.)</p>/n<ul>/n<li><a href="/wiki/SQL/CLI" title="SQL/CLI">SQL/CLI</a> (<a href="/wiki/Call_Level_Interface" title="Call Level Interface">Call Level Interface</a>) in 1995</li>/n<li><a href="/wiki/SQL/PSM" title="SQL/PSM">SQL/PSM</a> (<a href="/wiki/Stored_procedure" title="Stored procedure">stored procedures</a>) in 1996</li>/n</ul>/n<h2><span class="mw-headline" id="References">References</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=SQL-92&amp;action=edit&amp;section=3" title="Edit section: References">edit</a><span class="mw-editsection-bracket">]</span></span></h2>/n<div class="reflist" style="list-style-type: decimal;">/n<div class="mw-references-wrap">/n<ol class="references">/n<li id="cite_note-MeltonSimon1993-1"><span class="mw-cite-backlink"><b><a href="#cite_ref-MeltonSimon1993_1-0">^</a></b></span> <span class="reference-text"><cite class="citation book">Jim Melton;
